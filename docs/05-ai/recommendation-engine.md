# Recommendation Engine

## Purpose

Provide contextual, explainable recommendations from canonical editorial knowledge.

## Phase 1 Approach

Use editorially curated recommendation records and simple deterministic filters.

Do not build black-box ranking.

## Eligibility

An entity can be recommended only if:

- It has an approved editorial verdict.
- It has a rationale.
- It has confidence and verification status.
- It is not retired.
- Its caveats are known.

## Ranking Inputs

- Editorial score
- Relevant vertical score
- Confidence
- Freshness
- Context match
- Relationship strength
