# Page Types

## Home

Introduces the platform through locations, verticals, and timely editorial judgment.

## Location Page

Shows the best verticals, guides, maps, and canonical recommendations for a place.

## Location + Vertical Page

The core Phase 1 page type.

Examples:

- Big Sur Hotels
- San Gabriel Valley Boba
- Paso Robles Wine

Modules:

- Editorial intro
- Best overall
- Worth the splurge
- Hidden gem
- Map
- Comparison
- Caveats
- Related guides

## Place Page

Canonical record for a hotel, boba shop, winery, restaurant, or other place.

Modules:

- Verdict
- Scores
- Rationale
- Caveats
- Facts
- Map
- Related guides
- Nearby recommendations

## Guide Page

Ghost-authored editorial content with live canonical cards embedded from the database.

## Map View

Location-aware exploration with editorial filters and confidence indicators.

## AI Advisor

Conversational planning interface grounded in canonical knowledge.
