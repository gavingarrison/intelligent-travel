## Knowledge Layers

### Layer 1: Facts

Addresses, hours, coordinates, pricing, phone, website.

### Layer 2: Editorial Observations

Architecture, atmosphere, crowding, hospitality, texture, service, taste notes.

### Layer 3: Editorial Judgment

Verdicts, Halo, caveats, confidence, recommendations.

These layers remain independent.
