# Taste

Taste is modeled as structured editorial and user preference knowledge.

## Editorial Taste Dimensions

- Beauty
- Design Quality
- Authenticity
- Hospitality
- Atmosphere
- Service
- Food quality
- Family fit
- Sense of place
- Memory value

## User Taste Dimensions

- Boutique vs resort
- Quiet vs lively
- Luxury threshold
- Design focus
- Coffee interest
- Wine interest
- Children ages
- Walkability
- Adventure
- Food priority

## Rule

Taste can be modeled, but not fully automated. Human editors remain authoritative.
