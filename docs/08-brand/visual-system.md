# Visual System

## Direction

Premium editorial utility.

The interface should feel calm, legible, structured, and trustworthy.

## Product Requirements

- Location-first navigation
- Strong maps
- Clear place cards
- Visible verdicts and caveats
- Tasteful score display
- Editorial photography with rights metadata
- No OTA-style price-first layouts
- No cluttered review-site comparison tables

## Visual Priority

The user should see:

1. Where this is.
2. What it is.
3. Whether it is worth their time.
4. Why we believe that.
5. What caveats matter.
