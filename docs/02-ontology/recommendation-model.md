# Recommendation Model

Every recommendation stores reasoning.

## Recommendation Primitives

- Best Overall
- Worth the Splurge
- Hidden Gem
- Family Favorite
- Date Night
- Rainy Day
- Weekend Plan
- Nearby
- Similar To
- Better Than
- Halo Experience

## Required Recommendation Data

- Recommended entity
- Context
- Verdict
- Rationale
- Caveats
- Best for
- Not ideal for
- Confidence
- Verification status
- Evidence
- Last reviewed date

## Rule

A place can be known without being recommended.
