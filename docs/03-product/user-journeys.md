# User Journeys

## Choose A Hotel

1. User lands on Big Sur.
2. User selects Hotels.
3. User sees editorial verdicts, caveats, map, and comparison.
4. User opens a hotel card.
5. User understands whether it fits their trip.

## Find Boba

1. User lands on San Gabriel Valley Boba.
2. User filters by classic, fruit tea, cheese foam, seating, parking, or nearby.
3. User compares shops by taste notes and editorial confidence.
4. User saves a short list or opens map.

## Plan A Wine Day

1. User lands on Paso Robles Wine.
2. User selects style, pace, group, and food priority.
3. User receives a reviewed itinerary or AI-assisted plan grounded in canonical data.
4. User sees caveats, booking links, and driving context.
