# Narrative Leisure Intelligence Platform

This `/docs` system is the architectural constitution for the Narrative Leisure Intelligence Platform.

We are not building a travel blog. We are building an Editorial Intelligence Platform for leisure, hospitality, food, drink, travel, and meaningful time.

## Source Of Truth

The source PRD is `Narrative_Leisure_Intelligence_PRD_v2.md`.

## Platform Thesis

- Confidence is the product.
- Judgment compounds.
- Taste can be modeled.
- AI amplifies editors.
- Every recommendation stores rationale.
- Relationships are more valuable than records.
- Ghost is publishing, not the canonical database.
- The canonical knowledge graph is the company memory.
