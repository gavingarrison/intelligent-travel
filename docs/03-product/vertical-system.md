# Vertical System

Verticals are editorial lenses across locations.

## Initial Verticals

- Family Travel
- Hotels
- Restaurants
- Coffee
- Bakeries
- Wine
- Boba
- Parks
- Museums
- Family Experiences
- Seasonal Events
- Road Trips

## Vertical Requirements

Each vertical defines:

- Inclusion rules
- Relevant entity types
- Relevant scores
- Common recommendation primitives
- Filters
- Caveats
- Verification cadence

## Examples

### Boba

Important dimensions:

- Tea quality
- Texture
- Sweetness control
- Signature drinks
- Seating
- Parking
- Line/crowd patterns

### Wine

Important dimensions:

- Producer quality
- Tasting room experience
- Appointment rules
- Food pairing
- Driving context
- Hospitality
- Scenic value

### Hotels

Important dimensions:

- Service
- Design
- Location
- Family fit
- Value
- Luxury
- Booking complexity
