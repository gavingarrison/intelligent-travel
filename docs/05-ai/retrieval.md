# AI Retrieval

## Retrieval Stack

- Structured retrieval from canonical tables
- Semantic retrieval from embeddings
- Lexical search from search index
- Graph traversal from relationships

## Query Flow

1. Parse intent and constraints.
2. Retrieve candidate entities.
3. Filter by editorial eligibility.
4. Expand graph relationships.
5. Fetch canonical facts and rationale.
6. Generate answer with citations.
7. Expose confidence and caveats.

## Citation Requirements

AI answers should cite:

- Place records
- Guide records
- Recommendation records
- Score records
- Relationship records
- Last verification date
