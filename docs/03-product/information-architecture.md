# Information Architecture

## Primary Spine

The site is organized by regions and locations.

Examples:

- California
- Central Coast
- Big Sur
- San Gabriel Valley
- Paso Robles

## Vertical Lenses

Verticals organize what is worth knowing inside a location.

Examples:

- Big Sur Hotels
- San Gabriel Valley Boba
- Paso Robles Wine

## Top-Level Navigation

Phase 1:

- Locations
- Guides
- Maps
- Search

Phase 2+:

- Verticals
- AI Advisor
- Saved Places
- Membership

## Page URLs

Suggested patterns:

- `/locations/big-sur`
- `/locations/big-sur/hotels`
- `/locations/san-gabriel-valley/boba`
- `/locations/paso-robles/wine`
- `/guides/best-family-hotels-big-sur`
