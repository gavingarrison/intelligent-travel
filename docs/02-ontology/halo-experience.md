# Halo Experience

A Halo Experience is an experience with unusually high memory value, beauty, distinctiveness, and emotional recall.

## Required Fields

- Name
- Location
- Referenced entities
- Why it matters
- Best context
- Not ideal context
- Seasonality
- Friction
- Editorial confidence
- Verification status

## Examples

- A Big Sur coastal hotel stay anchored by views and architecture.
- A San Gabriel Valley boba crawl with a clear point of view.
- A Paso Robles wine day that avoids generic tasting-room fatigue.
