# Scoring Model

Scores support judgment. They do not replace it.

## Required Scores

- Editorial Score
- Design Score
- Service Score
- Food Score
- Beauty Score
- Family Score
- Value Score
- Luxury Score
- Halo Score
- Confidence Score
- Freshness
- Verification Status

## Score Record

Each score stores:

- Entity
- Score type
- Value
- Scale
- Label
- Rationale
- Methodology version
- Reviewer
- Evidence
- Visibility
- Validity window

## Public vs Internal

Public:

- Verdict
- Caveats
- Confidence label
- Last verified
- Selected scores where useful

Internal:

- Experimental score weights
- AI confidence
- Draft scores
- Ranking formulas
